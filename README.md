# Building AI Agents 🤖

## Overview
**Building AI Agents** is a growing collection of small-scale, modular AI agents designed to tackle specific tasks. This repository serves as a practical exploration of agentic architectures, demonstrating how to connect Large Language Models (LLMs) with external tools, APIs, and automation workflows. 

Rather than a single monolithic application, this project is broken down into lightweight, independent agents. Each folder contains a distinct agent with its own specific use case, ranging from simple content automation to multi-step reasoning.

## What's Inside?
This repository includes various types of agents, such as:
*   **Workflow Automators:** Agents that integrate with APIs (like YouTube or social platforms) to manage and publish content.
*   **Task-Specific Bots:** Focused scripts utilizing frameworks like Vertex AI or Hugging Face for specific NLP or generative tasks.
*   **Tool-Calling Agents:** Agents equipped with external functions (e.g., web scraping, database querying) to execute multi-step logic.
*   **Conversational Interfaces:** Simple chat agents tailored with custom system prompts and context retention.

## Tech Stack
*   **Language:** Python
*   **Models & Platforms:** (e.g., Langchain, LangGraph etc.)
*   **Automation:** (e.g., n8n, Botpress, custom REST API calls)
